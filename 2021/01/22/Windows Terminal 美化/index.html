Windows Ternimal 美化

Scoop 安装

步骤 1：在 PowerShell 中打开远程权限

步骤 2：自定义 Scoop 安装目录

可以不设置

如果跳过该步骤， Scoop将默认把所有用户安装的 App 和 Scoop 本身置于C:\Users\%username%\scoop

步骤 3：下载并安装 Scoop

步骤4：安装字体（可忽略）

Fantasque Sans Mono Nerd Font

步骤5. 安装oh-my-posh（类似oh-my-zsh）

步骤6：配置PROFILE文件

步骤6作用：PowerShell 每次打开的时候会还原为默认的样式，通过PROFILE文件可以使 PowerShell 每次启动时自动读取PROFILE文件的配置，自动应用主题

步骤7：主题选择

powerlevel10k-classic

图片

注意：由于部分字体的字符不全，需要将个别代码注释并修改掉，否则会显示异常

代码文档路径：C:\Users\%username%\Documents\WindowsPowerShell\Modules\oh-my-posh\2.0.496\ThemesPowerlevel10k-Classic.psm1

附：部分图标Unicode码

Markdown格式：&#x+unicode+;

⏰23F0 💨1F4A8 📂1C4C2

其他主题：agnosterplus、honukai、operator、powerline、powerlineplus、punk

Scoop 插件推荐

windows-screenfetch
